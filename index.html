制作自己的博客的方法

方法一：找一个大神帮忙
方法二：自己搜视频瞎琢磨
方法三：结合以上两种方法
{具体步骤：1：下载git并安装 2:下载node 3:注册GitHub 4：下载sublime text 3并安装 5：打开命令指令输入以下指令
【（1）npm install hexo-cli -g
（2）hexo init （blog）
（3）cd （blog）
（4）npm install
（5）hexo server
（6）npm install –save hexo-deployer-git】
6：下载编译器例如vs插件进行博客内容更改 7：在themes hexo网站下载主题zip并解压 8：在sublime text中改写即可}
（注意：1，命令界面不识别鼠标2，命令界面复制粘贴只能用快捷键3，命令界面打开方式windows+r再输入cmd）
